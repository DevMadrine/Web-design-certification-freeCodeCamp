Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

Example code: console.log('Hello World');

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. For example:

  • Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM). Example: document.getElementById('example');
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. Example: require('http');

Javascript and Java

JavaScript and Java are similar in some ways but fundamentally different in others. Example: if (true) { }

In contrast to Java's compile-time system of classes, JavaScript supports a runtime system based on a small number of data types.

Variables

You use variables as symbolic names for values. For example: let number = 10;

Example: var name = 'John';

Global variables

Global variables are in fact properties of the global object. Example: window.myVar = 100;

Another example: globalThis.example = 'test';
